Performance & Best Practices

Font Loading Strategies

  • System fonts: Instant loading, consistent across platforms
  • Preconnect: Used in this page for faster Google Fonts loading
  • font-display: swap: Shows fallback font while custom font loads
  • Fallback fonts: Always specify fallbacks in font-family

GDPR Considerations

Google Fonts can be GDPR problematic. Alternatives:

  • Self-host Google Fonts
  • Use Bunny Fonts (GDPR compliant)
  • Use system font stacks
