2. letter-spacing: Character Spacing Control

Normal letter spacing (default)

Tight letter spacing (-0.02em) - use carefully!

Wide spacing for headers (0.2em)

D R A M A T I C E F F E C T

letter-spacing: normal; /* Default */ letter-spacing: 0.1em; /* Wider spacing */ letter-spacing: -0.02em; /* Tighter spacing */ letter-spacing: 0.5em; /* Very wide for effect */

Use cases:

  • Headers and titles (slightly wider)
  • All-caps text (needs more spacing)
  • Logo text
  • Artistic effects

3. line-height: Line Spacing for Readability

Tight Line Height (1.0)

Normal Line Height (1.5) - Recommended

Loose Line Height (2.2)

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris. This can be good for certain artistic effects.

line-height: 1.0; /* Very tight */ line-height: 1.5; /* Good for body text */ line-height: 2.0; /* Loose, good for poetry */ line-height: 120%; /* Same as 1.2 */

4. text-transform: Case Conversion

Normal case remains unchanged

text-transform: uppercase; /* ALL CAPS */ text-transform: lowercase; /* all lowercase */ text-transform: capitalize; /* Title Case */ text-transform: none; /* No change */

Common uses:

  • Navigation menus (uppercase)
  • Buttons (uppercase)
  • Headings (capitalize)
  • Form labels

6. Text Overflow with Ellipsis

Single Line Ellipsis

This is a very long text that will be truncated with an ellipsis when it overflows the container

/* The three properties needed for ellipsis */ .ellipsis-text { white-space: nowrap; /* Prevent line breaks */ overflow: hidden; /* Hide overflowing content */ text-overflow: ellipsis; /* Add the ... */ }

Multi-line Ellipsis (Webkit only)

This is a longer text that demonstrates multi-line ellipsis. It will show the first few lines and then add ellipsis when the content overflows beyond the specified number of lines. This only works in Webkit browsers.

/* Multi-line ellipsis (Webkit only) */ .ellipsis-multiline { display: -webkit-box; -webkit-line-clamp: 3; -webkit-box-orient: vertical; overflow: hidden; text-overflow: ellipsis; }

üí° Pro Tips

  • Readability first: Don't sacrifice readability for style
  • Test on devices: Text effects may look different on various screens
  • Use relative units: Em and rem units scale better than pixels
  • Consider accessibility: Ensure sufficient contrast with text shadows
  • Performance: Complex text shadows can impact performance on older devices
