CSS Margin: auto and Collapsing Examples

margin: auto;
This box is centered both horizontally and vertically in its parent.
(Parent has fixed height)
margin: 0 auto;
This box is horizontally centered using margin: 0 auto; and a fixed width.
margin: 40px 100px;
Top and bottom margins are 40px, left and right are 100px.
The box is not centered, but offset from both sides.
margin: 0 0 0 auto;
Only left margin is auto, so the box sticks to the right edge.
(width: 200px)
margin: 0 auto 0 0;
Only right margin is auto, so the box sticks to the left edge.
(width: 200px)
