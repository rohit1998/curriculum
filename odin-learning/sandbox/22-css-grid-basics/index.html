Step 6: Named Grid Areas

Header
Sidebar
Main
Footer

Explanation: grid-template-areas lets you name and visually lay out areas.

/* CSS */
.step6 {
  display: grid;
  grid-template-columns: 120px 1fr;
  grid-template-rows: 40px 80px 40px;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}
.step6 .header { grid-area: header; }
.step6 .sidebar { grid-area: sidebar; }
.step6 .main { grid-area: main; }
.step6 .footer { grid-area: footer; }
