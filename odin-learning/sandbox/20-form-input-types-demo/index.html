Interactive Explanation

  • Text: For general text input.
  • Email: Validates email format, shows @ keyboard on mobile.
  • Password: Masks input for privacy.
  • Number: Only allows numbers, can set min/max.
  • Date: Shows a date picker.
  • Dropdown: Lets user pick one option from a list.
  • Radio Buttons: Pick one from a small set.
  • Checkboxes: Pick multiple options.
  • Textarea: For longer, multi-line text.
  • Submit Button: Sends form data.
  • Reset Button: Clears the form.
  • Generic Button: For custom actions (JS).

Try interacting with each input above to see how they work. Notice how labels help clarify what each input is for, and how some inputs (like email or number) have built-in validation or special UI features.
